Politique de confidentialité

Dernière mise à jour : 6 avril 2025

1. Introduction

La présente politique de confidentialité décrit comment Etudly ("nous", "notre" ou "nos") collecte, utilise et protège les informations que vous nous transmettez lorsque vous utilisez notre application et notre site web (collectivement désignés comme les "Services").

Notre équipe s'engage à respecter votre vie privée et à protéger les données personnelles que vous nous confiez. Cette politique vise à vous informer de manière transparente sur nos pratiques en matière de protection des données.

2. Qui sommes-nous

Etudly est une application éducative en phase de développement, créée par une petite équipe d'entrepreneurs et de développeurs. Nous nous efforçons de respecter les principes du Règlement Général sur la Protection des Données (RGPD) et de la Loi Informatique et Libertés.

Pour toute question concernant vos données, vous pouvez nous contacter à privacy@etudly.com.

3. Données que nous collectons

3.1 Données que vous nous fournissez directement

Lorsque vous utilisez nos Services, nous collectons les informations suivantes :

  • Informations de base : prénom, nom, adresse email
  • Informations académiques : établissement, niveau d'études, matières suivies, notes
  • Contenu utilisateur : documents d'études téléchargés, plans d'étude créés, résultats des questionnaires
  • Communications : messages que vous nous envoyez via notre formulaire de contact

3.2 Données collectées automatiquement

Lorsque vous accédez à nos Services, nos systèmes collectent automatiquement :

  • Données d'utilisation : pages visitées, fonctionnalités utilisées, temps passé sur l'application
  • Informations techniques : type d'appareil, système d'exploitation, navigateur
  • Données de connexion : adresse IP, date et heure d'accès

4. Comment nous utilisons vos données

Nous utilisons vos données personnelles pour :

  • Vous permettre d'accéder à nos Services et de les utiliser
  • Personnaliser votre expérience utilisateur
  • Améliorer notre application et corriger d'éventuels problèmes
  • Communiquer avec vous concernant nos Services
  • Comprendre comment nos utilisateurs interagissent avec l'application

5. Technologies que nous utilisons

Notre application utilise plusieurs technologies pour fonctionner :

  • Hébergement : Nous utilisons Vercel pour héberger notre site web
  • Développement : Notre code source est géré via GitHub
  • Services Google : Nous utilisons certains services Google pour l'analyse et le fonctionnement de l'application
  • Envoi d'emails : Nous utilisons des services d'envoi d'emails comme nodemailer

Ces services peuvent avoir accès à certaines de vos données dans le cadre de leur fonctionnement.

6. Partage de vos données

Nous ne vendons jamais vos données personnelles. Nous pouvons les partager dans les cas suivants :

  • Avec les prestataires techniques mentionnés ci-dessus, qui nous aident à faire fonctionner nos Services
  • Si nous y sommes contraints par la loi ou une décision de justice
  • Pour protéger la sécurité de nos utilisateurs ou de nos Services

Certains de ces prestataires peuvent être situés hors de l'Union Européenne, notamment aux États-Unis (GitHub, Vercel, Google). En utilisant nos Services, vous acceptez que vos données puissent être transférées vers ces pays.

7. Conservation des données

Nous conservons vos données aussi longtemps que nécessaire pour vous fournir nos Services :

  • Tant que vous utilisez activement votre compte
  • Jusqu'à 12 mois après votre dernière connexion
  • Plus longtemps uniquement si la loi nous l'impose

8. Sécurité de vos données

Nous prenons des mesures pour protéger vos données :

  • Utilisation de connexions sécurisées (HTTPS)
  • Stockage sécurisé des mots de passe
  • Limitation des accès aux données au sein de notre équipe

Cependant, aucun système n'est totalement sécurisé et nous ne pouvons garantir une protection absolue contre toutes les menaces potentielles.

9. Vos droits

Vous disposez de plusieurs droits concernant vos données personnelles :

  • Droit d'accès : obtenir une copie des données que nous détenons sur vous
  • Droit de rectification : corriger des informations inexactes
  • Droit à l'effacement : demander la suppression de vos données
  • Droit d'opposition : vous opposer au traitement de vos données
  • Droit à la portabilité : recevoir vos données dans un format réutilisable

Pour exercer ces droits, contactez-nous à @etudly.com. Nous traiterons votre demande dans un délai d'un mois.

10. Modifications de cette politique

Nous pouvons modifier cette politique de temps à autre pour refléter l'évolution de nos pratiques ou de nos Services. La version la plus récente sera toujours disponible sur notre site web avec la date de la dernière mise à jour.

Pour les modifications importantes, nous ferons notre possible pour vous en informer directement.
